¿Qué atributo se utiliza para especificar la URL a la que se enviarán los datos del formulario? a) method b) action c) target d) url

El atributo action de acción define la ubicación (URL) donde se deben enviar los datos recopilados del formulario cuando se envía

¿Cuál es el atributo que se utiliza para especificar el tipo de datos que se espera en un campo de entrada? a) data-type b) input-type c) type d) data

El atributo method define con qué método HTTP enviar los datos (generalmente get o post).
